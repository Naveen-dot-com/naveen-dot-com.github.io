Delete Your Oshowani Data

Oshowani stores all chat history only on your device — no data is stored on any server.

Option 1 — Delete from your Android phone:
Settings → Apps → Oshowani → Storage → Clear Data
Option 2 — Delete from browser:
Browser Settings → Privacy → Clear browsing data → Site data for oshowani
Option 3 — Email us:
Send a request to n.k.dubey1997@gmail.com
Subject: Data Deletion Request
We will respond within 7 days.
